Installation

Installation steps:

  1. Open dietpi.com select your SBC and download.

  2. Extract the zipped file.
  3. Using belenaEtcher flash OS image to target drive.
  4. Eject drive, plug into SBC and boot SBC. First boot takes a while.
  5. Find your device's local IP and connect via PuTTY .
  6. Login to your SBC:
    • login: root
    • password: dietpi
  7. Go through the steps on your screen and you're done!
